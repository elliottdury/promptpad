# Promptpad
## Project description
Promptpad aims to let users build powerful structured prompts for language and image models at speed.
## Architecture
Promptpad is built around several core components that work together to enable users to build powerful structured prompts for language and image models efficiently.
### Key Components
1. **Prompt Builder Interface**

    An intuitive web app interface for building prompt structures at any complexity

2. **Prompt Refinement Engine**

    An agentic framework that takes JSON model prompt structures and converts them into the appropriate format for a given model

3. **Template Library Engine**

    A process that fetches prompt templates from a given source assuming a given format.
